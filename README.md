# Hello Rails exercise

> This is an exercise that displays 'Hello World!' to the page by using [Rails](https://guides.rubyonrails.org/) to make sure everything is set up correctly.

<br>

## App created (with postgresql) by using the following command:

```
rails new hellorails --database=postgresql
```

## Install the pg gem:
```
gem install pg
```

## Install all the dependencies

```
bundle install
```

<br>

## Start the app:

```
rails server
```
